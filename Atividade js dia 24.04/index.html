Atalhos do VS code

Atalhos gerais

vamos ver alguns Atalhos legais do VS code que vao deixar a vida mais alegre, supinpa e feliz

alt + espaço: chama o auto completar

alt + z: quebra automatica de linha

alt + seta cima/ baixo: Move a seleção para cima/baixo no arquivo. se nada estiver selecionado, movimenta a linha em que o cursor esta

shift + alt + setas: copia a seleção para cima/baixo no arquivo. se nada estiver selecionado, copia a linha em que o cursor esta.

ctrl + ; : comentario/ descomentario
